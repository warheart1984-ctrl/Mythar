# Mythar ISF v0.4.0

Mythar ISF v0.4.0 ratifies the versioned Intermediate Semantic Form as the stable semantic IR for Compiler API v2.

## Included

- Ratified ISF v0.4 specification and JSON schema
- ISF conformance suite: 7/7 passing
- Preserved v0.2 (101/101) and v0.3 (5/5) conformance
- Governed English, Spanish, and Standard Mandarin contract format
- UTF-8-safe CLI output and semantic translation modes

## Constitutional boundary

The release ratifies the ISF contract, not a historical reconstruction, unrestricted machine translation, or a particular cloud platform. Language contracts remain independently versioned under the Transduction Contract Standard.

See the release manifest and ratification record for reproducibility and lineage.
