# Grammar and Lexicon

Canonical sentence order is Particle → Verb → Subject → Object → Modifier. Particles may stack; the first is governing.

v0.3 ratifies `ema` as the Root of beautiful existence, `fak` as the stackable Craft Particle, and `tila` as a postfix meta-operator that purifies/refines its preceding target.

Examples: `raema` is proclaimed beauty; `ja fak la ma` uses the craft frame; `maia tila` applies purity to the spring-goddess composite.
