# DOS Benchmark Suite v0.2

**Status:** Draft — Specified

v0.2 extends v0.1 with multi-artifact, cross-domain, and adversarial decision benchmarks. It measures graph depth, branching factor, invariant density, cross-domain continuity, fallback behavior, and challenge logs. No v0.2 result is prefilled.

| Domain | Purpose |
| --- | --- |
| D — Multi-artifact | AST + ISF + registry, grammar + migration, diagnostics + ratification |
| E — Cross-domain | Proposed interactions with CIEMS, CIVCA-X, and VOSS profiles |
| F — Adversarial | Malformed AST, conflicting lineage, ambiguity, contradictory evidence |

Metrics: graph depth is the maximum root-to-leaf distance; branching factor is edges/nodes; invariant density is checks/nodes. Cross-domain continuity requires evidence rather than an asserted relationship.
