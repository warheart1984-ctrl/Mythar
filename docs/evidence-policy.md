# CRS Evidence Policy

Mythar is a constitutional language-engineering and semantic-architecture project. It does not claim completed historical reconstruction.

Every major claim, lexeme note, grammar rationale, and research artifact must carry one of these classifications:

- **Observed** — supported by cited external evidence. Record sources, scope, and confidence.
- **Specified** — an explicit Mythar design decision. Record the governing registry/version and ratification authority.
- **Hypothesized** — a proposal for investigation. Record assumptions, falsifiers, and the path to review.

Versioned Zenodo deposits should include this policy, the relevant registry, conformance corpus, ratification records, API contract, and release notes. This keeps creative language design, comparative-linguistic research, and executable semantic engineering transparent and independently assessable.
