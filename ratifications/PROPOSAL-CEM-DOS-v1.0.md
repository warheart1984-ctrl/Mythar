# Proposal — CEM–DOS Integration Profile v1.0

**Status:** Pending review — not eligible for ratification

## Required evidence

- Green A01–C03 and D01–F03 benchmark results.
- Cross-runner replay evidence.
- Recorded conversational baseline runs using fixed prompts and settings.
- Complete benchmark matrix with evidence IDs, invariant outcomes, and release commit hashes.
- Demonstrated DOS Kernel invariant adherence.

## Current assessment

A01 has local three-run deterministic and typed-trace evidence. All other benchmark cases, cross-runner replay, and conversational comparisons remain pending. The CEM–DOS Integration Profile therefore remains **Specified v0.1** and is not promoted by this proposal.
