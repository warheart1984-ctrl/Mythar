# Mandarin Operator Alignment v0.4

**Status:** Draft for investigation  
**Evidence class:** Hypothesized

The following table is a proposed semantic teaching alignment only. It is not implemented by the Mandarin input adapter, does not claim that Mandarin particles and Mythar operators are equivalent, and MUST NOT be used as linguistic evidence without independent review.

| Mandarin particle | Approximate function | Proposed Mythar operator |
| --- | --- | --- |
| 了 (`le`) | completed aspect | `ta-` |
| 过 (`guò`) | experiential aspect | `ma-` |
| 着 (`zhe`) | continuous aspect | `fu-` |
| 吧 (`ba`) | suggestion | `ra-` |
| 呢 (`ne`) | continuation | `-wie` |
| 啊 (`a`) | emphasis | `ra-` |

Examples such as `ra + ta- → tara` and `la + fu- → fula` are proposals for future operator and clause conformance work, not current Mandarin compilation behavior.
