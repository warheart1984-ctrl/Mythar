# Mandarin to English through Mythar ISF v0.4

**Status:** Draft semantic-input demonstration

## Light

```text
Mandarin input: 光
Semantic adapter: 光 → la
ISF: la / illumination / truth / reveal
English transducer: illuminate
```

## Speak

```text
Mandarin input: 说
Semantic adapter: 说 → ra
ISF: ra / proclamation / speech / speak
English transducer: speak
```

The Mandarin adapter performs an explicitly specified semantic lookup; it is not Mandarin syntactic parsing or machine translation.
